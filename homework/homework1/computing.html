An interesting applicaiton of binary math is in classical computing. Computers
use binary numbers as the basis of all their calculations. This allows computers
to be a useful tool to people.
The reason for this is that at the base of a computer, it functions as an electronic
device. As electrical current can be harnessed most easily as on or off, this lends itself
to using a binary system of numerical representation. 1's represent on, and 0's
represent off. Now the user of a computer sees colorful images, hears articulate
sounds, and is able to interact with a variety of data and text. But again, all
if you could look deep under the hood of a computer, you would be able to see that
all you do on a computer is made possible by binary calculations.


Furthermore, in computing, data is often handled with a certain number of bits. This would be the
amount of places used. For instance:

  • 2 bits is represented by one digit: 1 or 0
  • 4 bits is represented by two digits: 11, 00, 10, or 01
  • 8 bits is represented by three digits: 000, 001, 010, 011, ... and so forth

As you can see, binary numbers are able to represent 2Nth, where N is the number of digits.
